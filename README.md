## Quick Started

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Custom

### Theme Colors

Change the color of the theme in `tailwind.config.js`.

Light mode | Dark mode
-----------|-----------
`main`     | `main-dark`
`sub`      | `sub-dark`
